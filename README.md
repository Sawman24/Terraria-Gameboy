# Terraria GBA

A Terraria clone for the Game Boy Advance, featuring world generation, mining, crafting, building, and combat.

## Controls

### General Gameplay
- **D-Pad Left/Right**: Move character left or right.
- **D-Pad Up**: Jump.
  - **Double Jump**: Press again in mid-air (requires **Cloud in a Bottle**).
  - **Flight**: Hold in mid-air for upward thrust (requires **Rocket Boots**).
- **A Button**: Interact with nearby chests (opens inventory/chest screen).
- **B Button**: Use the currently selected item (Sword, Pickaxe, Axe, Blocks, etc.).
  - **Teleport**: Hold while using the **Magic Mirror** to return to your spawn point.
- **L / R Buttons**: Cycle through the first 4 slots of your hotbar.
- **Select**: Toggle **Mining Mode**.
- **Start**: Toggle **Inventory / Crafting** screen.

### Mining Mode (Toggle with Select)
- **D-Pad**: Move the cursor around the player.
- **B Button**: Mine the block at the cursor (using Pickaxe/Axe) or place the selected block.
- **B Button (Hold)**: Continuous mining or placing.

### Inventory & Crafting (Toggle with Start)
- **D-Pad**: Navigate through item slots or the recipe list.
- **A Button**: 
  - Pick up, swap, or merge item stacks.
  - Craft the selected recipe (in Crafting mode).
- **Select**: Cycle between **Inventory**, **Crafting**, and **Chest Storage** (if a chest is open).
- **Start**: Close the screen.

### Title Screen
- **D-Pad Up/Down**: Select "Continue" or "New World".
- **A Button**: Confirm selection.

---

## Features
- **Dynamic World Generation**: Explorable worlds with caves, ores, and biomes.
- **Day/Night Cycle**: Real-time lighting shifts and window-based light masks for night exploration.
- **Save System**: Progress is automatically saved to SRAM (Battery Backup) when the player dies, or manually via the title screen.
- **Crafting**: Use Workbenches and Furnaces to craft torches, tools, and armor.
- **Combat**: Fight off slimes with swords and other weapons.
